# Slider

Slider user input.

![Slider example](../../media/examples/slider.png)


## Installation

    npm install @pencil.js/slider


## Examples

```js
import Slider from "@pencil.js/slider";

const options = {
    min: 0,
    max: 100,
    value: 42
};
const slider = new Slider(aPosition, options);
```
