# EventEmitter

Ellipse shape (oval).


## Installation

    npm install @pencil.js/ellipse


## Examples

```js
import Ellipse from "@pencil.js/ellipse";

const horizontalRadius = 50;
const verticalRadius = 20;
const options = {
    // ...
};
const ellipse = new Ellipse(aPosition, horizontalRadius, verticalRadius, options);
```
